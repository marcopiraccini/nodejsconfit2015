KNX

  • KNX (Konnex) is a standardized (EN 50090, ISO/IEC 14543), OSI-based network communications protocol for intelligent buildings.
  • In KNX architecture devices are connected with a BUS
  • Sensors gather information and send it on the BUS as a data telegram. These can be information about room temperatures, movements, wind measurements or manually input instructions (e.g. swtiches).
  • Actuators receive data telegrams which are then converted into actions. Toggle ligths, open windows, turn on heating system,...

...so it's a message-based architecture, that uses a common bus for communication
